jValid, Easy form validation with jQuery

jQuery Form validation plugin

jValid is an easy configurable form validation plugin for jQuery.

Input visual feedback

The form fields will change their color depending on its status, and you can edit it from css.




Basic field validation

The plugin allows you to valid:

  • Required Fields
  • Optional Fields (this OR this)
  • Optional Fields (this AND this)
  • Min length
  • Max length
  • *Fancy* field names
  • Default field validation functions (input, email, file, textarea, url, date, chars, varchars, decimal)

How? simple!

<form id="form-id"> <input name="input-name1"> <input name="input-name2"> <input type="submit"> </form> <script> $(document).ready(function() { $("#form-id").jvalid({ rules : { 'input-name1' : [ true , 'input' , 'Fancy Name' , '', '', -1, -1 ], 'input-name2' : [ false , 'email' , 'Fancy Name' , '', '', -1, -1 ] } }); }); </script>

The ´rules´ parameter contains the rules that apply to each input with the next order:

'INPUT NAME' : [REQUIRED, 'VALIDATIONFUNCTION', 'FANCYNAME', 'AND-INPUT-NAME', 'OR-INPUT-NAME', MIN-LENGTH, MAX-LENGTH]

Custom field validation functions

Need a custom validation function? Maybe a REGEX?, Just pass the function as an argument!

$("#formID").jvalid({ rules : { 'inputName' : [ true , 'customFunction' , 'Fancy Name' , '', '', -1, -1 ], }, 'customFunction' : function(txt){ if(txt == '') return true; var customRegex = /^[0-9]?$/; return customRegex.test(txt); } });

Custom error messages

Need custom error messages? Maybe change language?, Just pass any of the next parameters to overwrite the error messages ;)

$("#formID").jvalid({ rules : { 'inputName' : [ true , 'customFunction' , 'Fancy Name' , '', '', -1, -1 ], }, inputRequired_msg : 'The field \'%s\' is required', inputMinLength_msg : 'The field \'%s\' cannot contain less than %d characters', inputMaxLength_msg : 'The field \'%s\' cannot contain more than %d characters', inputBothEqual_msg : 'The fields \'%s\' and \'%d\' have to be equal', inputEitherSel_msg : 'Fields \'%s\' or \'%d\', either one is required', inputBadSyntax_msg : 'The field \'%s\' contain syntax errors' });

Custom actions

Want to change how error messages are shown? or do something before the form is submited? or maybe don't submit and do an AJAX call instead? Just overwrite the show, hide and submit functions!

$("#formID").jvalid({ rules : { 'inputName' : [ true , 'customFunction' , 'Fancy Name' , '', '', -1, -1 ], }, show : function(msg) { alert(msg); }, hide : function( ) { // Hide msg if needed }, submit : function( ) { // If everything is ok, what to do? return true; }, });
